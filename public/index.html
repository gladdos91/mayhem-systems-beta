Commands

40+ Slash Commands

Full autocomplete. No prefixes. Everything works from both Discord and the web panel.

🎫 Tickets (OpenTicket)
/ticket panel
Create button or dropdown ticket panel
/ticket category
Add category with admin + readonly roles
/ticket question
Add modal question to a category
/ticket priority
Set none / low / medium / high
/ticket move
Move ticket to a different category
/ticket transfer
Transfer ownership to another user
/ticket blacklist
Block users from opening tickets
/ticket stats
Global or per-user ticket statistics
/ticket transcript
Generate full HTML transcript
/ticket config
Set autoclose, cooldown, limits

/voice lock

/voice unlock
Unlock your voice channel

/voice limit

/voice permit
Allow a user into a locked channel
/voice reject
Remove and block a user
/voice claim
Claim an ownerless temp channel
/voice info
