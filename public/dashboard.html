Back up and restore all bot settings for this server

📤 Export Configuration

Downloads a complete JSON backup of every module's settings — ticket panels, categories, questions, automod rules, welcome config, auto roles, reaction roles, rules panels, server logs, and temp voice config.

Does not include: ticket message history, transcripts, or automod warning records.

📥 Import Configuration

Restore settings from a previously exported JSON file. This will overwrite existing settings for any module included in the file.

ℹ️ Ticket message history and Discord messages (panels, reaction roles, rules) are not affected — only database settings are restored.
